# BMO

BMO is a Java task chatbot for managing todos, deadlines, and events.

## Run

- GUI entry point: `bmo.Launcher`
- CLI entry point: `bmo.Bmo`

## Build and Test

- Run tests with `./gradlew test`

## Documentation

- User guide: `docs/README.md`
